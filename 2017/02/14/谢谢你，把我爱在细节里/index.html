谢谢你，把我爱在细节里

发表于 2017-02-14   |   分类于 记事本   |     |  


我总是一如既往地对你嫌弃不已，

而你却还是一往情深地对我只褒不贬。

我们的关系，

并未因距离而生疏，

反而越发亲密。

你总是说，

你越来越爱我了，

我是相信你的。

因为你不仅爱在语言里，

也爱在行动里，

爱在小事里，

爱在细节里。

谢谢你。




走在马路边，

你总会习惯性地走在靠近车道的一边；

过马路的时候，

你也会紧紧挽着我的手。

你会记得我平时不经意说出口的需求，

然后在某个重要或不重要的节日里，

突然给我惊喜。




你还总是对我的生活观察入微，

比我更了解我需要添置什么物品，

然后悄悄地就给我网购，

直到我收到了快递，

我知道，这又是你的惊喜。




听说别人的男朋友，

总会要求女朋友穿裙子，穿高跟鞋，涂指甲油，浓妆艳抹……

这样能让他们比较有面子。

而你，却觉得简单得体舒适就最好。

知道穿高跟鞋脚不舒服，

所以你只会给我买舒适的运动鞋；

看到由于涂指甲油导致的惨案的新闻，

你会心疼地说，

“亲爱的，咱们不要涂指甲油。”




在我痛经翻来覆去的时候，

你会给我买暖宝宝，

会给我煮红糖姜茶；

当你看到好看的电视剧，

你会买好会员推荐我去看；

当你看到又好看的电影，

会告诉我很好看，

但不推荐我看，

担心我一个人看会被吓到。




当我们一起短途旅游的时候，

我会安排好所有的行程；

当我被一些小事琐事惹得心烦意乱的时候，

我可以把它们丢给你帮我解决。




我是一个做事容易投入的人，

经常做着一件事，

就忘了看手机的新消息，

也忘了告诉你我正在做着某件事，

突然手机就来了一个微信视频，

我才猛然发现，

我又做错事了。

而你却从来都不生气，

说只是确认一下我没事就好。

我经常换位思考，

如果是我是你，

我该会多生气啊。




谢谢你，

把我的小事当作大事来处理；

谢谢你，

注意着我生活中的各种细节；

谢谢你，

对我的包容与爱。




情人节，你说：送你鞋子，希望和你一直走下去。

我已无语凝噎：T_T


一生漫长,只有住在对方心里,才能感受到在乎和爱啊!——《找一个把你的小事当大事的人很重要》。
